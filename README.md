## 开发指南

### 工具版本

- node 版本 `v10.16.0`
- npm 版本 `v6.12.0`

### 开发命令

1. 运行 `npm install` 安装依赖包
2. 运行 `npm run start` 启动 dev 服务器，支持命令行：
   - `--env=prod` 指定 `process.env.BUILD_ENV` 环境
   - `--no-hmr` 禁用热加载功能
3. 运行 `npm run build` 编译发布包
4. 运行 `npm run build:prod` 编译生产环境包
